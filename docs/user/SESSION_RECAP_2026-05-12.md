## TOP 10 IDEAS — Merged across all three AIs

1. **ARIA preserves meaning, not memory** — the thesis. Everything else is
   an expression of this. Memory = storage. Meaning = interpretation protected
   against flattening.

2. **This-defeats-the-purpose detector** — before building or saving any
   protocol, ask: what would technically satisfy this but defeat its purpose?

3. **Convergence Trap** — easy consensus among all three AIs = smell, not
   green light. Hard-won consensus = signal. Invoke Skeptic when it's easy.

4. **Working Memory Layer** — 5 items, visible always: current base task /
   parked thread / active decision / one fragile intent / next action.
   USER_DASHBOARD is already trying to be this.

5. **Skeptic + Champion paired-precise** — Skeptic names ONE breaking
   assumption. Champion names ONE strongest reason to keep it. Precision
   is the idea, not the roles.

6. **Idea Half-Life** — without aging, compost becomes graveyard you stop
   trusting. 60 days without promotion → "still relevant?" → promote, kill,
   or defer (max 3 deferrals).

7. **Sync Gap Detector + Sync Packet** — ARIA notices when one AI falls
   behind. Generates lightweight catch-up using repo/commit log, not token-
   heavy paste.

8. **Mind-Change Receipt** — track not just decisions but what changed your
   mind. Initial position → challenge → what changed → new decision → lesson.

9. **Overnight Safety Tier Ladder** — read-only (inventory what exists) →
   digest (synthesize what matters) → build (modify files). These are
   different operations for different needs, not just safety levels.

10. **ARIA = Uncrumple structurally** — both route messy evidence to usable
    decisions. Different domains, identical architecture. Every Uncrumple
    design decision is an ARIA design decision.

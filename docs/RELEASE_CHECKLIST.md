### 6. Camera permission is not declared in app.json

File: app.json

The app uses `expo-image-picker` (`launchCameraAsync`) in app/client-detail.tsx. expo-image-picker auto-injects `CAMERA` permission on Android, but Play Store now requires explicit declaration of all sensitive permissions and a justification string.

Recommended additions to `app.json` under `expo`:
```
"plugins": [
  ...,
  ["expo-image-picker", {
    "cameraPermission": "Uncrumple uses the camera to capture receipt photos."
  }]
]
```

Without this, the runtime permission prompt will show a generic system message, and Play Console may flag the camera permission as unjustified.

### 9. Date is stored as a free-form string

File: store.tsx, lines 23–28

Transaction `date` is a string like `'Apr 14'`, `'Today'`, `'Mar 28'`. There is no year, no ISO format, no sortable timestamp.

Implications for real users:
- Cannot sort transactions reliably across years.
- Tax tab cannot filter by year (currently hardcoded to whatever the date string says).
- Search and Recently Edited filters cannot work correctly.
- Cross-year invoice history will be ambiguous (Mar 28 of which year?).

This is a data-model issue, not a "release blocker" in the strict sense — the app does not crash. But it will become a customer-support problem within months of launch.

Do NOT fix this casually as part of another patch (per CLAUDE.md store.tsx rules). Plan a dedicated migration: add `dateISO` field, backfill for existing users, update display logic.

### 10. `receipt` boolean and `receiptUri` are out of sync in seed data

Files: store.tsx (seed), CLAUDE.md (Receipt Rules section)

Seed transactions set `receipt: true/false` but never set `receiptUri`. The cleanup flow uses `receiptUri` as source of truth. Result: seed transactions with `receipt: true` are still flagged as "Missing Receipt" in cleanup filters.

If seed data is removed (item 1), this issue is moot at first launch. If seed data stays, this is visibly broken on every fresh install.

CLAUDE.md explicitly warns against fixing this casually. Plan a coordinated cleanup if it must be fixed.

## Dev Workflow

---

🤖 Stop hook auto-copies Claude's last response to the Windows clipboard — fires every time Claude finishes, no button press needed. Just press paste wherever you want it. Tradeoff: clipboard gets overwritten on every response, so anything you copied mid-task is lost.

---

🤖 Backup option: button-pad macro runs a PowerShell one-liner that reads the latest transcript file under .claude/projects/ and copies the last assistant message on demand — preserves the clipboard between presses.

---

🤖 v0.dev and similar tools generate working React components from plain English descriptions — could speed up UI work vs hand-writing StyleSheets.

---

🤖 AI image generation for app assets — icons, splash screens, Play Store marketing images generated from prompts instead of hiring a designer.

---

🏗️ HTML prototype first workflow — build throwaway HTML mockups with AI to validate UX ideas before committing to React Native code.

---

🤖 Cursor IDE — VS Code alternative with inline AI editing built in, popular with vibe coders. Worth comparing against Claude Code workflow.

---

🏗️ Expo Snack — test UI ideas in the browser without touching your real project. Validate layouts before letting Code touch client-detail.tsx.

---

🏗️ Figma + AI plugins — design screens visually first, convert to code after. Faster than building UI by feel in StyleSheet.

---

🤖 NativeWind — Tailwind CSS for React Native. Replaces hand-written StyleSheets with utility classes. Polished components much faster.

---

🤖 Tamagui — React Native UI component library with built-in animations and theming. Would have made Uncrumple's dark theme trivial to implement.

---

🏗️ GitHub Copilot in VS Code — inline suggestions as you type, sees your whole codebase. Already have VS Code open, worth enabling.

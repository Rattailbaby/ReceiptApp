## Tips

---

💡 Win+L instantly locks your Windows PC — useful when you walk away from your desk while Claude Code is still running.

---

💡 Alt+Tab switches between open windows on Windows — hold Alt and keep pressing Tab to cycle through them, release to jump to that window.

---

💡 In PowerShell, pressing the up arrow key cycles through your previous commands — so you don't have to retype git add . every time.

---

⭐ IMPORTANT
💡 Power tip: Settings → System → Power & Sleep → set Screen to turn off after 1-2 minutes and Sleep to Never — screen goes dark to save energy but Claude Code and Remote Control keep running in the background.

---

💡 You can see how many tokens a Claude Code session has used by typing /usage in the Claude Code terminal.

---

🏗️ If you find yourself doing the same manual step more than three times, that's a sign it should be automated — your "wrap session" command is a perfect example of that thinking.
